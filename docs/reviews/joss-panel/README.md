# JOSS manuscript panel

This directory records a simulated, multi-role editorial review of `paper.md`.
It is an internal quality exercise, not a JOSS editorial decision and not
evidence of independent human adoption.

The review proceeds from the whole submission to individual sentences:

1. scope, significance, and pre-review screening;
2. paper purpose, audience, argument, and required-section structure;
3. section and paragraph function;
4. scientific, technical, impact, and comparative claims;
5. citation and evidence support;
6. sentence-level clarity, precision, tone, and mechanics;
7. rendered-paper and repository-package checks.

Each role writes a separate report. A synthesis records agreements, conflicts,
priorities, and accepted or rejected recommendations. After revisions, the same
roles reassess the complete manuscript rather than reviewing only the changed
sentences.

The scoring protocol is defined in [rubric.md](rubric.md). Submission
eligibility is tracked separately because no manuscript score can establish
JOSS's demonstrated-research-impact gate. The same-author `vop_poc_nz`
integration is the current research-workflow evidence; non-author activity is
a strong positive signal, not a separate universal gate.
