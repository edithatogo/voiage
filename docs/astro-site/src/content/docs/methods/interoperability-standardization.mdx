---
title: Interoperability and standardization VOI
description: Estimate the value of harmonizing evidence for common-data-model and cross-site reuse.
---

The fixture-backed `value_of_interoperability_standardization` method evaluates
schema compatibility, semantic alignment, data usability, evidence reuse,
transformation-error reduction, and standardization cost. It reports
standardization value separately from the baseline evidence utility.

The method remains `fixture-backed` while licensed cross-site data, open-data
provenance, Rust/binding parity, and mature/stable governance approval are
outstanding.

Use `voiage generate-config interoperability-standardization` to create a
starting JSON payload, then run
`voiage calculate-interoperability-standardization INPUT.json`.
