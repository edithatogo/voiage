---
title: Explainability and transparency VOI
description: Estimate the value of information from interpretable and transparent model evidence.
---

The fixture-backed `value_of_explainability_transparency` method separates
predictive utility from transparency-driven model adoption and governance value.
It combines explanation quality, transparency evidence, and trust into an
adoption score, then accounts for governance impact and audit cost.

The method is intentionally marked `fixture-backed`. External model validation,
licensed open-model evidence, Rust/binding parity, and mature/stable approval
remain required before promotion.

Use `voiage generate-config explainability-transparency` to create a starting
JSON payload, then run `voiage calculate-explainability-transparency INPUT.json`.
