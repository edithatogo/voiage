---
title: Capacity and budget-constrained VOI
description: Value information when feasible decisions are limited by budget and capacity.
---

The `value_of_capacity_budget_constrained` method compares the best strategy
under resource constraints with the strategy selected when the scenario is
known. It reports budget impact, capacity shortfall, constrained regret, and
shadow-price diagnostics.

The current implementation is fixture-backed. Licensed allocation traces,
Rust parity, and stable-promotion review remain explicit gates.
