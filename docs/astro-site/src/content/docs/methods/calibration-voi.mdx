---
title: Calibration VOI
description: Value of Information for Model Calibration
---

Calibration VOI captures the value of reducing uncertainty through model calibration, surrogate fitting, or related evidence-alignment work. This method is relevant when the decision problem depends on parameters that can be estimated by fitting the model to observed data.
