---
title: Equity-information VOI
description: Fixture-backed information value for uncertain equity weights
---

`value_of_equity_information` quantifies the value of resolving uncertain
equity weights. It accepts subgroup net benefits, baseline weights, plausible
post-acquisition scenarios, scenario probabilities, and information cost.

This optional extension is distinct from the experimental distributional-
equity API. Stable promotion requires Rust and retained-binding parity,
licensed open-data attribution, and mature/stable approval.
