---
title: Network NMA
description: Network Meta-Analysis Value of Information
---

Network Meta-Analysis VOI extends traditional VOI analysis to decision problems where multiple treatments must be compared simultaneously through a connected evidence network. It quantifies the value of reducing uncertainty in relative treatment effects across the entire network.
