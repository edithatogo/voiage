---
title: Structural VOI
description: Value of Information for model structure uncertainty
---

Structural VOI quantifies the value of reducing uncertainty about the model structure itself, rather than only the input parameters. This family addresses decision problems where multiple competing model forms (e.g., different disease pathways, statistical specifications, or evidence networks) each carry non-negligible plausibility.
