---
title: Strategic behavior and game-theoretic VOI
description: Estimate information value when other actors respond strategically.
---

The fixture-backed `value_of_strategic_behavior` method models equilibrium
assumptions, incentive response, disclosure value, bargaining value, strategic
regret, response sensitivity, and adversarial response.

Calibrated game data, cross-language parity, and mature/stable governance
approval remain required before promotion.

Use `voiage generate-config strategic-behavior` to create a starting JSON
payload, then run `voiage calculate-strategic-behavior INPUT.json`.
