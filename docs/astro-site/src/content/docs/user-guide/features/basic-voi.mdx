This guide covers the fundamental VOI metrics: Expected Value of Perfect Information (EVPI) and Expected Value of Partial Perfect Information (EVPPI).

See the [canonical basic VOI markdown guide](https://github.com/edithatogo/voiage/blob/main/docs/user_guide/features/basic_voi.md) on GitHub for the full reference.

## EVPI

The Expected Value of Perfect Information (EVPI) is the maximum amount one should be willing to pay to eliminate all uncertainty. It provides an upper bound on the value of any further research.

## EVPPI

The Expected Value of Partial Perfect Information (EVPPI) identifies which parameters or parameter subsets drive decision uncertainty. It helps prioritize research efforts toward the most impactful uncertainties.
