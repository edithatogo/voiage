---
title: Metamodeling
description: Surrogate modeling for VOI analysis
---

Metamodeling (surrogate modeling) in VOI analysis uses statistical or machine learning models to approximate complex simulation models, making computationally intensive VOI calculations tractable.

voiage provides metamodeling support through regression-based EVPPI methods and Gaussian process regression for EVSI estimation.
