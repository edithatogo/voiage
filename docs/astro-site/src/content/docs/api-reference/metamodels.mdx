---
title: Metamodel API
description: Optional metamodel and acceleration surface reference
---

Metamodels and accelerator integrations are optional Python extensions. They
must not be imported by the minimal Rust/Python installation and do not replace
the Rust numerical authority.

Use the extension policy and the relevant optional dependency extra before
constructing a metamodel. Stable result envelopes and validation contracts
remain unchanged when an optional backend is unavailable.
