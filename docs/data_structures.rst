Data Structures
===============

`voiage` uses a small set of explicit data structures to keep method inputs and
outputs serializable, inspectable, and stable across bindings.

Key structures:

- `ValueArray`
- `DecisionAnalysis`
- `Intervention`
- `DecisionProblem`
- `TrialDesign`
- `ParameterSet`
