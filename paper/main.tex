\begin{abstract}
Value of Information (VOI) analysis estimates how much a decision could
improve if uncertainty were reduced by collecting additional information.
\texttt{voiage} is open-source research software for computing, validating,
and communicating VOI analyses from probabilistic decision models. Version
1.0.0 combines a Python user interface with authoritative Rust domain,
numerical, diagnostic, and serialization components. It implements expected
value of perfect information, expected value of partial perfect information,
expected value of sample information, expected net benefit of sampling,
cost-effectiveness acceptability frontiers, dominance, heterogeneity, and
structural-uncertainty methods. PyO3 exposes the Rust core to Python; a
versioned C application binary interface supports direct EVPI calls from R and
Julia. Language-neutral fixtures, explicit result schemas, structured errors,
and layered assurance gates make the stable numerical contract testable across
interfaces. This paper describes the statistical scope, architecture, and
evidence boundaries of the v1.0.0 release.
\end{abstract}
