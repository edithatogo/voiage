\begin{abstract}
Parameter-focused Value of Information (VOI) analysis does not by itself
represent uncertainty about model structure, population heterogeneity,
analytical perspective, implementation, evidence quality, or the timing and
reversibility of action. These dimensions can affect assessments of further
research, validation, subgroup analysis, implementation support, or delayed
commitment. The \texttt{voiage} package implements shared data and result
structures for these linked problems. Stable interfaces cover classical
perfect, partial-perfect, and sample information; expected net benefit of
sampling; dominance; cost-effectiveness acceptability frontiers; and
heterogeneity, while their estimator paths retain distinct assumptions and
validation limits. Maturity-labelled extensions formulate value questions for
structural uncertainty, equity-weighted subgroup aggregation, perspective
disagreement, imperfect implementation, adaptive and sequential studies,
causal transportability, data quality, model validation, and staged
irreversible decisions. The
software contribution described here is an executable framework that records
strategy, sample, subgroup, perspective, population, and provenance semantics
across methods; it is not a new statistical formulation of classical VOI\@.
Diagnostics and result contracts record selected assumptions and computation
metadata. This paper describes the problem classes, estimands, domain
applications, and evidence boundaries of version 1.0.0.
\end{abstract}
