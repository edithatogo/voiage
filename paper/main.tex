\begin{abstract}
Value of Information (VOI) analysis is often reduced to parameter uncertainty,
although real decisions also depend on model structure, population
heterogeneity, analytical perspective, implementation, evidence quality, and
the timing and reversibility of action. These dimensions determine whether
further research, validation, subgrouping, implementation support, or delayed
commitment is worthwhile. \texttt{voiage} provides a common decision-model
framework for these linked problems. Its stable methods cover expected value
of perfect, partial-perfect, and sample information; expected net benefit of
sampling; dominance; cost-effectiveness acceptability frontiers; heterogeneity;
and structural uncertainty. Maturity-labelled extensions formulate value
questions for distributional equity, perspective disagreement, imperfect
implementation, adaptive and sequential studies, causal transportability,
data quality, model validation, and staged irreversible decisions. The
distinctive contribution is not a new label for classical VOI, but an
executable framework that preserves strategy, sample, subgroup, perspective,
population, and provenance semantics across methods. Explicit diagnostics and
result contracts expose assumptions that otherwise remain hidden in
one-off analyses. This paper defines the problem classes, estimands, domain
applications, and evidence boundaries of version 1.0.0.
\end{abstract}
