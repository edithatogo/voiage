\section{Glossary}\label{sec:glossary}

\begin{description}
  \item[Decision strategy] An available action or intervention whose expected
  value is compared with alternatives.
  \item[Net benefit] A common decision-value scale combining relevant outcomes
  and costs so that strategies can be compared.
  \item[Parameter uncertainty] Uncertainty about quantities within a specified
  model.
  \item[Structural uncertainty] Uncertainty about the form, assumptions, or
  included relationships of a decision model.
  \item[Implementation uncertainty] Uncertainty about whether adoption,
  adherence, coverage, delay, or execution will realize modeled value.
  \item[Analytical perspective] The viewpoint determining which outcomes,
  costs, populations, and time horizons enter a decision.
  \item[Transportability] The extent to which evidence from a source setting or
  population can inform the target decision.
  \item[Result envelope] A structured return value containing an estimate
  together with method, maturity, diagnostics, scaling, and provenance
  information.
  \item[Maturity label] An evidence classification distinguishing stable,
  fixture-backed, and experimental method families.
\end{description}
