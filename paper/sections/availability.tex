\section*{Code and data availability}

The analysis uses \texttt{voiage} version 1.0.0, random seed 20260723 for the
probabilistic analysis, and seed 20260724 for the bootstrap. The worked example
is generated by
\path{scripts/generate_paper_health_example.py}; all parameter draws and
figure data are synthetic and are recreated by that script. The plotted
study-value results and prespecified sensitivity scenarios are also provided
as machine-readable CSV files in \path{paper/data/}. Source code, release
artefacts, and checksums are available at
\url{https://github.com/edithatogo/voiage/releases/tag/v1.0.0}. The repository
snapshot is archived by Software Heritage as
\url{swh:1:snp:767efde24c97d9f6d730764c1b3bc1a91ba20c32}.
