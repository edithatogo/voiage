\section{Artificial intelligence usage disclosure}

OpenAI Codex and Google Jules generative artificial intelligence (AI) tools
assisted with repository analysis, implementation review, security and release
workflow review, documentation drafting, and manuscript preparation. The
repository does not retain a complete retrospective inventory of model
versions used across development. AI-produced suggestions were reviewed
against source code, tests, references, and generated artifacts; independent
review agents were also used to identify statistical, arXiv, and journal
readiness issues. The human author remains responsible for the software
design, claims, references, author list, and final text. No AI tool is an
author or a substitute for scientific and editorial judgment.
