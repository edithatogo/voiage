# arXiv paper skill

Build with `uv run task arxiv-readiness`. Inspect the canonical PDF, LaTeX
source archive, SHA-256 digest, semantic HTML, and readiness evidence. Never
upload or infer an arXiv identifier. Category, license, author, and final-upload
decisions remain human gates.
