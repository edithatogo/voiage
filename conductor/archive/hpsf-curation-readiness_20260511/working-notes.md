# Working Notes: HPSF Curation Readiness

## Focus

- HPSF is tracked as a curation visibility channel; users should not assume install
  readiness from HPSF listing alone.
- Evidence currently required: release metadata, license, versioned docs, stable
  package identity, and maintainer contact/ownership.

## Handoff Notes

- HPSF receives explicit curation metadata and provenance references, not package
  uploads from this repository.
- Acceptance is external and depends on the curation review process.

## External dependency

Submission and approval workflow remains external to this repository.
