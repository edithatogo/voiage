# Monitoring-Surveillance Fixtures

This directory defines the fixture-backed contract for periodic monitoring,
signal detection, stopping rules, and decision revision value. The Python
runtime and CLI are fixture-backed; open-data attribution, cross-language
parity, and stable promotion remain explicit gates.

The runtime accepts deterministic synthetic panels with dimensions
`samples x strategies x periods` and reports cost, delay, false-signal,
detection, revision, and stopping diagnostics.
