# Core API Schemas v1

This directory holds the versioned JSON Schemas for the stable v1 contract.

The stable v1 contract matrix currently includes:

- `intervention.schema.json`
- `decision-problem.schema.json`
- `trial-design.schema.json`
- `parameter-set.schema.json`
- `value-array.schema.json`

Stable result schemas:

- `results/ceac.schema.json`
- `results/enbs.schema.json`
- `results/evpi.schema.json`
- `results/evppi.schema.json`
- `results/evsi.schema.json`

Semantic rules that cannot be expressed cleanly in JSON Schema live in the track foundation document at `../../foundation.md`.
